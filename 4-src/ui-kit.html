Dropzone Styling

Dropzones appear when dragging tasks or lists to indicate valid drop targets. Choose a color that provides clear visual feedback without being distracting.

Option 1: Subtle Grey

Uses grey-30 (#EBEBE5) background with grey-60 (#BCBCBC) border. Neutral and unobtrusive, blends well with the design system.

Drop zone indicator
background-color: rgba(235, 235, 229, 0.5); /* grey-30 with 50% opacity */
border: 2px dashed #BCBCBC; /* grey-60 */

Option 2: Light Blue

Selected

Uses blue-500 (#6B8FD9) with very light opacity (4%) for background and dashed border. Matches the focus ring color, providing visual consistency with interactive elements.

Drop zone indicator
background-color: rgba(107, 143, 217, 0.04); /* blue-500 with 4% opacity */
border: 2px dashed rgba(107, 143, 217, 0.4); /* blue-500 with 40% opacity */

Option 3: Very Light Grey

Uses grey-20 (#F0F0EE) background with grey-50 (#DBDBD5) border. Minimal and subtle, provides the least visual distraction.

Drop zone indicator
background-color: #F0F0EE; /* grey-20 */
border: 2px dashed #DBDBD5; /* grey-50 */
